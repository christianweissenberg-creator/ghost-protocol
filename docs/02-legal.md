# Rechtliche Findings — Deutschland

> [!danger] Dieses Dokument ersetzt keine Rechtsberatung. Bei Unsicherheit: Steuerberater/Anwalt konsultieren.

## Prediction Markets
**Status: ❌ ILLEGAL in Deutschland**
- GGL klassifiziert Polymarket als illegales Glücksspiel (GlüStV 2021)
- §284 StGB: Veranstaltung illegalen Glücksspiels — strafbar
- §285 StGB: Teilnahme an illegalem Glücksspiel — strafbar
- Polymarket sperrt DE-IPs, Trading explizit verboten
- Kalshi: nur für US-Residents
- Keine legale Prediction-Market-Alternative in DE verfügbar
- **Entscheidung: Komplett gestrichen** → [[06-decisions]]

## Krypto-Trading via Bots
**Status: ✅ LEGAL (eigenes Konto auf regulierten CEXs)**
- Trading auf BaFin-lizenzierten/MiCA-konformen Exchanges erlaubt
- 3Commas, eigene Bots via API = legal
- BaFin-Warnung betrifft nur UNLIZENZIERTE Bot-Plattformen
- Steuerpflicht: Gewinne <12 Mo Haltedauer + >€1.000/Jahr → Einkommensteuer
- DAC8 ab 2026: automatische Meldung an Finanzamt

## Affiliate-Marketing Krypto
**Status: ✅ LEGAL (mit Auflagen)**
- Werbung muss als solche gekennzeichnet sein
- Risiko-Disclaimer erforderlich
- Keine Garantie-Versprechen
- MiCA Marketing-Regeln beachten

- Keine persönliche Anlageberatung
- Disclaimer auf jeder Seite/Mail

## Gewerbeanmeldung

- Keine BaFin-Lizenz nötig (kein Verwalten fremder Gelder)
